## ğŸ“‹ Project Overview

**Yulu** is India's leading micro-mobility service provider, offering unique vehicles for the daily commute to eliminate traffic congestion. This project is a comprehensive data analysis case study built to understand the factors affecting the demand for these shared electric cycles in the Indian market.

This interactive **Streamlit Dashboard** performs detailed **Exploratory Data Analysis (EDA)** and rigorous **Hypothesis Testing** to uncover demand drivers, seasonal patterns, and the impact of weather conditions on bike rentals.

## âš™ï¸ Installation & Usage

Follow these steps to run the project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Ratnesh-181998/Yulu-Bike-Sharing-Demand-Analytics.git
    cd Yulu-Bike-Sharing-Demand-Analytics
    ```

2.  **Create a virtual environment (optional but recommended):**
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    ```

3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Run the Streamlit app:**
    ```bash
    streamlit run app.py
    ```
